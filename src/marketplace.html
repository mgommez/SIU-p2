Comandos de voz para interactuar con el sistema

"Título libro" para que se añada a tu galería personal el libro con dicho título. Recuerda, debe ser uno de los títulos mostrados en esta pantalla.

"Cancelar" para finalizar el proceso. No se añadirá ningún libro y se volverá a la pantalla de galería personal.

Comandos de voz para interactuar con el sistema

"Añadir libro" para añadir a tu galería personal un libro de los disponibles en el marketplace de la aplicación.

"Borrar libro" para eliminar algún libro de tu galería personal.

"Información libro" para consultar la información del libro.

"Seleccionar libro" para seleccionar alguno de los libros.
